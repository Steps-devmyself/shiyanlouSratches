如何从亚马逊/Opencart/WooCommerce等平台导入导出产品到Shopify？

作者: Alex是大叔 分类: Shopify小技巧,Shopify独立站操作笔记 发布时间: 2018-12-08 14:50

自从Shopify火了以后，各种厂家纷纷效仿推出了电商自建站的产品-BigCommerce、WooCommerce、Magento、3dcart等等。就连之前的拖拽建站网站：Weebly，Wix等都进来插了一脚推出了电商建站服务。市场充分竞争对于我们广大电商从业者来说其实是好事，但是随着业务不断扩大、我们的SKU越来越多，一旦有了更换平台的想法就会变得非常麻烦。比如我已经做了亚马逊，想建Shopify独立站，或者我已经做Shopify很成功，现在不想花大量手续费，想自己搭建服务器转移到Opencart，但是有大量SKU需要手动传。。那么你就需要这个工具：

Cart2Cart

这是我自己在后期遇到的一个问题：有了技术以后，我已经不想受Shopify的平台约束，特别是支付方面，想转移到完全开源的Opencart，但是面对几百上千个SKU无从下手，于是通过搜索发现了这个网站-Cart2Cart，在抱着怀疑的态度试用了以后完全被折服，确实厉害，果断付费，等了不到10分钟，所有产品准确无误的导入成功。。

看了一眼官方支持的平台：

厉害了，基本上市面上的所有电商平台都可以互相导入导出，这次我测试的是Shopify >> Opencart，具体流程如下：

注册完成之后，到这个页面选择需要导入导出的平台：

上边是要导出的平台，下边是要导入的平台。

Shopify API Password是需要到Shopify >> APP创建一个新的APP并赋予所有权限，然后把API输入到这里。

导入到Opencart需要在服务器端上传一个插件，具体看网站上讲的很清楚，如果有不懂的可以留言。

接下来选择需要转移的产品属性：评论、分类、税、运费、历史订单、优惠券等等

下边附加收费的按需选择，一般用不到

接下来会免费的做一个测试，大概会上传20个产品左右

确认产品上传无误之后，付费即可上传所有产品

点击这里可以获得估价：

1000个产品以内最低$69美元，1000以上根据产品数量决定价格。个人感觉还不错，花钱买时间，而且上传的产品不会出错，值得推荐。
